# 🐾 にゃんタスク管理アプリ

##  概要
PHPとMySQLを使用して開発したタスク管理アプリです。  
「今日のタスク」に特化し、未完了タスクを上部に固定表示し、
達成率をリアルタイムで可視化できるように設計しました。

##  主な機能
- タスク追加 / 編集 / 削除
- 完了切り替え機能
- 今日のタスクを上部固定表示
- 今日のタスクのみ達成率を算出
- カレンダー表示（月単位）
- PDOを使用した安全なDB接続
- ログイン、ログアウト機能の追加
##  使用技術
- PHP
- MySQL
- PDO
- HTML / CSS
- XAMPP（ローカル開発環境）

##  工夫した点
- 今日の日付のみを抽出し、優先表示ロジックを実装
- 達成率バーを動的計算
- ORDER BY を活用し未完了タスクを上部に表示
- UIを見やすく柔らかいデザインに調整

##  データベース設計

### tasks テーブル
| カラム名 | 型 | 説明 |
| id | int | 主キー |
| task | varchar | タスク内容 |
| task_date | date | 実行日 |
| is_done | boolean | 完了フラグ |
